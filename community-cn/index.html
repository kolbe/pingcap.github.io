Events
TechDay

TiDB TechDay 是以城市为主体的技术分享活动，社区开发者共同学习、分享 TiDB 技术内核，目前已成功在上海和深圳举办。
DevCon

TiDB 开发者大会是由 TiDB 官方筹备并组织的年度大会，垂直面向开发者。讨论的焦点是 TiDB 技术最新发布的特性、最佳实践以及 TiDB 社区运营经验，旨在促进社区成员间的深度交流。
Hackathon

TiDB Hackathon 是由 PingCAP 举办的黑客马拉松大赛，选手需围绕整个 TiDB 生态做出一个完整的作品并由评审最终评选出获奖者。参赛队伍需在规定时间（48 小时）内完成作品并展示。

社区阅读

这个板块将实时更新源码阅读系列以及社区小伙伴撰写的新手指南。 源码系列将从整体架构到重要模块抽丝剥茧，深入浅出的为大家解析 TiDB 设计思路。新手指南则会手把手教你成为 TiDB contributor。 ​感谢社区成员对系列教程的大力支持。
